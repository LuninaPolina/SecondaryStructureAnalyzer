# 16s rRNA detection

## Binary classification: 16s VS other genomic subsequences

  * Sequence length = 512
  * Model was trained on vectorized data: train:valid:test = 21322:10234:7039

## Results on test set

 * TP = 2965, TN = 3400, FP = 65, FN = 609 
 * accuracy = 90.4%
